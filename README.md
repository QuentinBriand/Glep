## What is libglep ?

Glep is a library that aims to provide a simple and easy to use library for OpenGL. It is written in C++ and made to be structured like the SFML library but for 3D rendering. It is currently in development and is not yet usable.

## Project hierarchy

I followed this very good article to structure my project : [Canonical Project Structure](https://www.open-std.org/jtc1/sc22/wg21/docs/papers/2018/p1204r0.html)
